&#x20;**Service Enumeration**



Purpose



\-Identify service details and versions







\## Command



bash

\-nmap -sC -sV -p <ports> 192.168.100.10



What You Get

* Service name
* Version
* Default scripts output

Example



\-Port 445 → SMB → possible enumeration



Key Insight



\-Knowing service version helps find vulnerabilities
