\# Red Teaming Journey 🚀



This repository documents my structured journey into \*\*Red Teaming, Penetration Testing, and Ethical Hacking\*\*.



The focus is not just on tools, but on:



\* Understanding systems deeply

\* Developing attacker mindset

\* Chaining vulnerabilities into full compromise



\---



\## 🎯 Objectives



\* Build a strong foundation in networking, Linux, and Windows

\* Learn real-world offensive techniques

\* Practice in isolated lab environments

\* Document methodology and findings professionally



\---



\## 🧪 Lab Environment



\* Kali Linux (Attacker)

\* Windows 10 (Target)

\* VMware Workstation

\* Network Setup:



&#x20; \* NAT (Internet access)

&#x20; \* Host-only (Internal attack network)



\---



\## 🧠 Learning Approach



Instead of blindly using tools, I follow this methodology:



```

Recon → Enumeration → Analysis → Exploitation → Privilege Escalation

```



\---



\## 📂 Repository Structure



\* \*\*01-lab-setup/\*\* → Environment setup \& troubleshooting

\* \*\*02-core-foundations/\*\* → Essential knowledge

\* \*\*03-recon-and-scanning/\*\* → Nmap \& discovery techniques

\* \*\*04-enumeration/\*\* → Service-level attacks (SMB, RPC, etc.)



\---



\## ⚠️ Disclaimer



All activities are performed in a \*\*controlled lab environment\*\* for educational purposes only.
